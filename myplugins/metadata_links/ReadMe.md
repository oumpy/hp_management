# MetadataLinks Plugin for Pelican
## What is it ?
This plugin enables you to use Pelican link conventions, such as {filename} or {'attach'}, in metadata of articles & pages.

## Settings
`METADATALINKS_FIELDS`: list of metadata fields, whose values are links for which this plugin should be applied.
